# Solana Wallet Dashboard

A modern, high-performance web application for interacting with the Solana blockchain. This project provides a seamless interface for managing Solana assets, tracking balances in real-time, and performing secure SOL transfers.

## Key Features

*   **Premium UI Design**: A glassmorphic, dark-themed interface built using modern CSS principles and the Inter typeface for a high-end Web3 experience.
*   **Real-Time Synchronization**: Automatic UI updates for account balances utilizing Solana's onAccountChange WebSocket listeners for zero-latency state management.
*   **Secure Transaction Handling**: Full integration with the official Solana Wallet Adapter for delegated signing, ensuring that private keys never leave the user's wallet.
*   **Performance Optimized**: Leveraging Vite and React for near-instant load times and a highly responsive user experience.
*   **Devnet Integrated**: Pre-configured for the Solana Devnet cluster, allowing for safe testing and development of blockchain interactions.

## Technical Stack

*   **React**: A declarative, component-based library for building a modular and maintainable user interface.
*   **Solana Web3.js**: The core library used for blockchain interactions, account management, and building transaction instructions.
*   **Solana Wallet Adapter**: A unified interface for connecting to Phantom, Solflare, and other major Solana wallet providers.
*   **Vite**: A modern build tool utilized for high-performance development and optimized production deployments.

## Getting Started

1.  **Install Dependencies**: `npm install`
2.  **Run Development Server**: `npm run dev`
3.  **Build for Production**: `npm run build`
